ccask_get_result* ccask_db_get ( ccask_db *  db,
uint32_t  key_size,
uint8_t *  key 
)

get implementation 1) try to get kdrow from keydir (ccask_keydir_get(...)) 2) from kdrow, we additionally get value_size, file_id, value_pos 3) now from our file pointer, we seek to value_pos.

4) read 32 bit crc 5) read 32 bit tstamp, 32bit ksz, 32bit valuesz, key_size bytes key, value_size bytes value. 6) calc crc of values from (5) 7) compare read crc and calc'd crc 8) return the value & crc result.
